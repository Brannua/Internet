例如，当我们在浏览器的地址栏中输入某个网站的域名后，就可以访问该网站的内容，这就是推动因特网飞速发展的万维网应用 WWW，其相关的应用层协议为超文本传送协议 HTTP

用户在地址栏中输入的是“见名知意”的域名

而 TCP/IP 体系的网际层使用 IP 地址来标识目的主机

从域名到 IP 地址的转换工作，由属于应用层的域名系统 DNS 在后台帮用户自动完成，以方便用户的使用
